# GDIvsDWrite
A demo for comparing GDI vs DirectWrite font rendering.
![](assets/compare.png)

## Usage
Just download and extract the zip in [Github release page](https://github.com/HO-COOH/GDIvsDWrite/releases) and run `GDIvsDWrite.exe`.
